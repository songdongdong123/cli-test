### 本地执行
1. clone代码到本地后，在当前跟目录下，执行npm link
2. 执行fe
3. 控制台输出 ‘hello ethan-cli‘

### package.json中bin字段解释
bin 是一个在 package.json 文件中的属性，用于指定一个或多个可执行文件的路径。当你在全局安装一个包时，这些可执行文件将被添加到系统的 PATH 环境变量中，以便在命令行中直接使用。
